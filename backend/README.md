## Pour modifier le backend
Le backend tourne sur CodeWords. Pour le modifier:
1. Ouvrir CodeWords (https://codewords.agemo.ai)
2. Trouver le service `budget_app_api_b2ecbb91`
3. Modifier et deployer

Le fichier `budget_app_api.py` dans ce repo est une copie de reference.
Il doit etre mis a jour manuellement apres chaque modification du backend.
